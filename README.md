# 💖 3 anos de namoro

Um pequeno projeto feito para celebrar **3 anos de namoro**.
A ideia é criar uma página especial com uma **galeria de fotos**, animações e um visual bonito para lembrar momentos importantes.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap

---

## 💡 Objetivo

Esse projeto foi criado como uma forma simples e criativa de **celebrar momentos importantes em um relacionamento**, utilizando programação e design web.
